OPSEC WARNING

Upgrading to session mode opens a persistent TCP connection for real-time C2. This is detectable:

  • Visible in netstat as ESTABLISHED
  • Detectable by EDR network monitoring
  • Persistent connection (not polling)
  • Traceable if connection drops mid-op

HTTP beacon polling is stealthier. Use session mode when you need real-time response.
